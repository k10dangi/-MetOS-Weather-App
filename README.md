# MetOS Intelligence - Futuristic Weather Dashboard

MetOS Intelligence is a high-performance, cyberpunk-inspired weather telemetry system built with React and TypeScript. It provides advanced atmospheric insights through a sleek, tactical interface.

## 🚀 Key Features

- **Real-Time Atmospheric Telemetry**: Precision monitoring of temperature, humidity, wind velocity, and barometric pressure.
- **Satellite Link**: Integrated live satellite imagery and geographic data via Leaflet and Esri.
- **Temporal Projection Scrubber**: Advanced simulation tool to visualize weather trends up to 24 hours into the future.
- **Environmental Safety Monitoring**: Real-time AQI and UV Index tracking with risk assessment.
- **Health Intelligence**: Integrated AI-driven health recommendations based on current environmental conditions.
- **Dynamic Visualizations**: Animated trend charts for temperature and precipitation saturation using Recharts.
- **Night-Ops Mode**: Toggleable dark theme optimized for low-light environments.

## 🛠 Tech Stack

- **Framework**: React 18
- **Language**: TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS & Custom Design System
- **Icons**: Lucide React
- **Maps**: React-Leaflet
- **Charts**: Recharts

## 📥 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/MetOS-Weather-App.git
   ```

2. Navigate to the project directory:
   ```bash
   cd MetOS-Weather-App
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

## ⚙️ Configuration

MetOS operates with internal mock data by default for immediate deployment. To activate live satellite uplinks from OpenWeatherMap:

1. Obtain an API key from [OpenWeatherMap](https://openweathermap.org/api).
2. Click the **Config API** button in the dashboard.
3. Input your key and initialize the system.
